# Sistema-de-inventario-clinico
Sistema completo de inventario para clínicas que permite gestionar medicamentos y materiales con control de stock, caducidad y trazabilidad de movimientos. Diseñado para optimizar procesos y reducir errores en el manejo de insumos médicos.
